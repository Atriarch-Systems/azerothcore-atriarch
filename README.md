# azerothcore-atriarch

Corresponding Source for the Atriarch AzerothCore private server (WotLK 3.3.5a),
published to satisfy the GNU AGPL v3 network-use provisions of AzerothCore and
its modules.

This tree is a vendored snapshot: the AzerothCore Playerbot-branch core
(mod-playerbots project) plus the modules compiled into the server binaries,
including local modifications (weighted bot gender generation, Ollama LLM chat
and bot-control integrations, director-mode bot AI).

Each release of the running server pushes a commit here tagged with the
server's image version. Licenses: see LICENSE (GNU AGPL v3 / GPL v2 per
upstream components).

Upstream projects: https://www.azerothcore.org - https://github.com/mod-playerbots
